# Big data processing tutorials

## Prerequisites 

Install [pixi](https://pixi.sh/latest/)

## Installation

```bash
git clone --recurse-submodules https://github.com/esarrazin/big-data-processing-course-tutorials.git
cd big-data-processing-course-tutorials
pixi install 
```

## Launch jupyterlab

```bash
pixi shell
jupyter lab
```
